## Copy Guidelines

✓ **Always Include:**
- Real numbers ($0.07, 1440 calls, 6.61/10)
- Time comparison (24h vs 2-4 weeks)
- Open source mention
- GitHub link

✗ **Avoid:**
- Hype ("revolutionary", "state-of-the-art")
- Unverified claims
- Comparisons to named competitors
- Unreleased features

**Tone:**
- Honest (real data, not polished hype)
- Humble (built by developer for developers)
- Practical (outcomes, not buzzwords)
- Community-friendly (invite feedback)
