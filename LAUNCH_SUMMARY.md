# Prompt Engineer Skill — Launch Summary

## What You're Launching

An open-source Claude Code skill that automates prompt optimization at scale.

**In One Sentence:**
Run 1000+ prompt combinations in a day instead of hand-testing 5-10. Get a deployment-ready winning prompt with cost and confidence data.

**In Two Paragraphs:**

Most teams optimize prompts by hand. They write 5-10 variations, test on 100-200 examples, cherry-pick the best, and deploy. It takes 2-4 weeks and often fails in production. There has to be a better way.

Prompt Engineer Skill automates this entire process. You describe your goal. The skill researches available LLMs and techniques, builds a combinatorial experiment matrix (templates × parameters × models), runs hundreds or thousands of API calls, evaluates every result using 4 independent methods, and delivers a ranked leaderboard with the winning prompt ready to deploy. The entire process takes 24 hours and costs less than $1. Works with Groq, OpenAI, Anthropic, Ollama, or any OpenAI-compatible provider.
